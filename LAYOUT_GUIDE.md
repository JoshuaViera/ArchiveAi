## Animation Timings

| Element | Duration | Delay | Effect |
|---------|----------|-------|--------|
| Dashboard container | - | 0.2s | Stagger children 0.1s |
| StatCards | 0.4s | 0s | Slide up + fade |
| Quick action cards | 0.3s | 0.3s-0.45s | Scale + fade |
| Voice profile card | 0.4s | 0.4s | Slide up + fade |
| Generate split panels | 0.4s | 0s-0.1s | Slide in from sides |
| Confidence meter | - | 0.3s | Width animation |
| Format recommendations | 0.4s | 0.2s | Slide up + fade |
| Generated result | 0.3s | - | Scale up + fade |
| Hover effects | 0.2s | - | Scale/color change |
| Button tap | 0.15s | - | Scale down to 0.95 |

## Color Scheme Preserved

- **Accent**: `#7C6AEF` → `#9B8DF5` (gradient)
- **Surface**: `#1a1d2e` (base)
- **Surface-2**: `#222534` (alternate)
- **Text Primary**: `#e4e5ea`
- **Text Muted**: `#a8aab5`
- **Text Dim**: `#878ba0`
- **Border**: `#262936`
- **Success**: `#34D399` (emerald)
- **Warning**: `#FBBF24` (amber)

## Responsive Breakpoints Used

- **Mobile**: Default (< 640px)
- **Tablet** (`md:`): 768px+
- **Desktop** (`lg:`): 1024px+

All text sizes, spacing, and layouts adapt automatically.
